# student-management-system
A student management system project built using java
## Description
A Java-based Student Management System that helps manage student records efficiently.

## Features
- Add and manage student details
- Store and retrieve student data
- Simple console-based interface
- Written using core Java and OOP concepts

## Tech Stack
- Language: Java
- IDE: VS Code
- Data Storage: File / Excel

## How to Run
1. Clone this repository
2. Open the project in VS Code or any Java IDE
3. Compile and run `SMS.java`
